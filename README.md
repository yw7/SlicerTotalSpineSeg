# SlicerTotalSpineSeg

Fully automatic total spine instance segmentation in 3D Slicer using "TotalSpineSeg" AI model

## Overview

This extension provides a 3D Slicer interface to the [TotalSpineSeg](https://github.com/neuropoly/totalspineseg) deep learning model for automatic spine segmentation.

## Installation

Install the extension from the 3D Slicer Extension Manager.

## Usage

1. Load your MR or CT spine image into 3D Slicer
2. Open the "Total Spine Seg" module
3. Select your input volume
4. Click "Apply" to run the segmentation
